HtmlEditor

介绍

富文本编辑器，（extjs中嵌入百度Ueditor编辑器）

示例代码

Ext.define('App.demo.form.field.HtmlEditor',{
	
	init: function(){
		var me = this;
		var view = Ext.create('Ins.form.Form',{
			fields : [{
					//label
					text:"富文本编辑器",
					name:"richText",
					xtype:"ins_htmleditor",
					value:'富文本默认值',
					width:600,
					height:300,
					//获取焦点事件 me:Ext.Component。
					onFieldFocus : function(me){
						console.log("获取焦点事件");
						console.log(me);
						//setValue返回值是富文本对象本身即me。
//						var returnMe = me.setValue("富文本内容");
						alert(a);
//						alert("给富文本赋值：富文本内容"+me.setValue("富文本内容"));
					},
					//失去焦点事件 me:Ext.Component;
					onFieldBlur : function(me){
						console.log("失去焦点事件");
						console.log(me);
//						alert("获取富文本赋值:"+me.getValue());
					},
					//改变内容事件  me:Ext.Component;
					onFieldChange : function(me){
//						console.log("改变内容事件");
//						console.log(me);
					},
					//验证提示消息默认采用side效果：显示在右边 其他效果有 qtip,title,under,none,[element id]
					msgTarget: 'side'
			}],
			actionButtons : [{
				text:'获取富文本对象并获得其嵌入的ueditor对象',
				handler: function(){
					var obj = view.getField("richText");
					//UEditor编辑器对象。
					//其他用法请查看官方Api文档：http://ueditor.baidu.com/doc/;
					//在线示例：http://ueditor.baidu.com/website/onlinedemo.html
					var UEditor = obj.ueditorInstance;
					var context = obj.ueditorInstance.getContent();
					alert(context);
				}
			}]
		});
		return view;
	}
});

类名

Ins.form.field.HtmlEditor

别名

ins_htmleditor

继承自

Ext.form.FieldContainer

构造对象

{
	text:"富文本编辑器",
	name:"name",
	xtype:"ins_htmleditor",
	value:'富文本默认值',
	width:600,
	height:300
}

属性

属性名属性值类型描述默认值
text string 输入框标签名称
name String 与后台实体类属性对应
xtype String 对象类型 ins_textfield
value String 默认值
ueditorInstance Ueditor对象 百度编辑器对象，该属性可理解为类的静态属性，由构造器初始化，直接通过Ueditor.ueditorInstance调用即可。
msgTarget String 验证不通过消息提示位置（qtip,title,under,side,none,[element id]） side

方法

返回值方法名称描述
me setValue("富文本内容") 给富文本赋值，参数String类型，返回值HtmlEditor对象，注意：该方法会覆盖富文本中原来的值
void getValue() 返回富文本内容带标签格式
void setReadOnly(boolean flag) 设置为只读

事件

返回值事件名称绑定方法描述
void onFieldFocus function(me){ } 获取焦点事件 me:Ext.Component
void onFieldBlur function(me){ } 失去焦点事件 me:Ext.Component
void onFieldChange function(me){ } 改变内容事件 me:Ext.Component;
